FAQ

How do I install CTGP-7 on my Nintendo 3DS?

To install CTGP-7, visit its GameBanana page.

Does CTGP-7 work on Citra3DS emulator?

You will need to repack the romfs. However, you will not have access to features like CC Selector or Updates and we do not provide assistance for Citra.

Where can I find the download for the CTGP-7 Installer?

The CTGP-7 installer download can be found here.

Does CTGP-7 work with Local Play?

Yes, CTGP-7 does work with Local Play. However, all 3DS must either A: Have the mod installed or B: Have one 3DS system with the mod installed and the rest connect with Download Play. Keep in mind, only custom tracks will work, not custom characters or mods unless method A is being used.

All of my save data is gone in CTGP-7, but not Mario Kart 7. What do I do?

You'll need to backup your save data using a Save Data Manager like Checkpoint. Then you'll need to copy the backed up save data in the SD: CTGP-7/savefs/ folder.

CTGP-7 crashes whenever I load the game or press a button.

If CTGP-7 crashes upon loading or pressing a button, you most likely have a custom font installed and need to uninstall it. Most custom fonts are made correctly and neither Mario Kart 7 nor CTGP-7 support custom fonts.

How do I report a bug?

You can report a bug on the CTGP-7 Discord Server.

CTGP-7 Launcher says my Zip file is invalid, how do I fix this?

If the launcher says your CTGP-7.zip file is invalid, try downloading the file again and make sure not to repack the file. You may also encounter this error if you are using a Mac.

How do I report a bug?

You can report a bug on the CTGP-7 Discord Server.

How do I report a bug?

You can report a bug on the CTGP-7 Discord Server.

How can I make a suggestion?

You can make a feature suggesstion on the CTGP-7 Discord Server. Please note that your feature may not make it into a release for many reasons.

Does CTGP-7 have multiple language support?

Yes, CTGP-7 does have support for languages like German and Spanish.

How can I contribute to CTGP-7?

You can contribute to CTGP-7 in many ways like making custom tracks and custom characters. If you have anything to contribute visit the CTGP-7 Discord Server

How can I contribute to this FAQ?

You can open a pull request or an issue on GitHub.

My question/issue isn't answered here.
