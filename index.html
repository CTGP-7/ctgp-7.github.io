What is CTGP-7?

CTGP-7 is a modpack for Mario Kart 7 which adds new things to the game, such as custom tracks and characters, extra features and more.

Check the full list of features here.

How do I install CTGP-7?

First of all, make sure you have installed Custom Firmware in your 3DS by following this guide.

Then, visit the GameBanana Page to download the mod and check the Installation PDF for the instructions.

Alternatively, you can use the following installation tutorial video.

If you need help installing Custom Firmware visit The Homebrew Community and if you need help with CTGP-7 visit the CTGP-7 Discord Server

Can I install CTGP-7 without a PC?

You can install the mod directly from the 3DS if you already have Custom Firmware. Check Method 2 on the Installation PDF to learn how. Keep in mind that this method can take up to 2 hours to install.

Do I need Mario Kart 7 to play CTGP-7?

Yes, you need a copy of Mario Kart 7 and the v1.2 update patch for both playing on Citra Emulator and on a 3DS.

We do not support piracy, you may be denied support if you ask for help with pirating 3DS games in our Discord or any other Homebrew communities.

How can I get Mario Kart 7 and the v1.2 update?

The eshop has closed, however if you've bought Mario Kart 7 in the past on the eshop you will be able to redownload Mario Kart 7, The ability to redownload has not been shutdown by Nintendo. Otherwise you will need a copy of Mario Kart 7 on a cartridge, used cartridges can be bought online or in local stores.

To obtain the v1.2 update you simply need to have your 3DS connected to the Internet, open Mario Kart 7 and you should be prompted to update, if you do not recieve this visit the eshop and search for "Mario Kart 7 Update" and download from here.

We do not support piracy and you can be denied assistance if you choose to pirate software.

Does CTGP-7 work on Citra emulator?

Yes! You can dump your cartridge or digital game as well as the update data and install them to Citra as a .cia file. Finally, install the CTGP-7 installer .cia file (using File -> Install cia).

What is the language and region support CTGP-7 offers?

CTGP-7 Supports USA, European, Japanese and Korean copies of Mario Kart 7 and the v1.2 update that matches your region of choice, There is currently no support for Chinese/Tawianese copies of Mario Kart 7.

CTGP-7 Supports mulitiple languages thanks to the help of the MK Translation Project and its team.

We currently support English, Català, Deutsch, Nederlands, Français, Magyar, Italiano, 日本語, 한국어, Português, Русский, Español.

All these languages are offered region free and are preinstalled with the modpack, if you find errors within the translations or would like to contribute to a language missing please visit the Mario Kart Translation Discord.

Does CTGP-7 work in local multiplayer?

Yes, CTGP-7 does work with Local Play. However, every console must be running CTGP-7. Keep in mind that Download Play doesn't work.

Selecting Create Group in local multiplayer freezes my console.

This is a bug with Mario Kart 7 games running from cartridge on Old 3DS models. The only known workaround for this issue is to install the cartridge as a cia to the SD. Check this guide to learn how to do so.

My save data is gone in CTGP-7! What do I do?

Mario Kart 7 and CTGP-7 save files are independent so that the Mario Kart 7 save file remains untouched. Follow this video guide to transfer your save data from MK7 to CTGP-7.

How do I stream using NTR with CTGP-7?

CTGP-7 uses a modified version of NTR. If you want to stream via NTR, press X in the CTGP-7 launcher. Remember not to press Close when returning to the home menu, press the Home button again instead.

How do I add my own stuff?
(Characters, music, karts, etc)

You can use the MyStuff folder to add your own cosmetic mods. Read this guide to learn how.

The 3DS online servers are shutting down, what is next?

The 3DS online servers have been announced to shut down on April 8th, 2024. Likewise, the fucntionality of Mario Kart 7 online will be shutdown, including World Wides, Races, Battle Mode, Ghosts, Communities, etc.

CTGP-7 has an already established a replacement network for CTWW, Countdown, World Wide and Battle Mode, online rankings and stats are saved within the CTGP-7 Network, the network is also moderated to prevent cheating and other unfair play.

For other 3DS online games you should visit the Pretendo Network website for more information about the fan made replacement service for 3DS and Wii U.

Why can I not connect to Mario Kart 7 online anymore and can CTGP-7 help me?

Nintendo has slowly started to shutdown or limit functionality of the 3DS servers in preparations for the April 8th shutdown. As a result some users have reported an inability to connect to Mario Kart 7 online.

You can play on Nintendo Network with CTGP-7 but it may not necessarily fix your problems, you can also play regular Worldwide races and Battle Modes on the CTGP-7 Network, with the benefits of a better anti-cheat.

The most common problem people have is their DNS server not being able to find a functioning Nintendo server. Changing your DNS settings in the system settings may help. click here to learn how to change your DNS.

For some however it is un-explainable why you may be having issues, aside from Nintendo's apathy/incompetence.
